Remact.Net.Bms1Serializer
=========================

**BMS1** is a binary data serialization format.

Remact.Net.Bms1Serializer is a C# implementation of the [BMS1 specification](http://github.com/steforster/bms1-binary-message-stream-format).

It is developed to be portable on .NET and MONO platforms.

Version 0.1: Initial development.
